# Developer Setup

This directory contains snippets, configurations, and scripts for setting up local development environments and workstations.

## Contents

- **`zsh-prompt.md`**: A custom two-line `zsh` prompt configuration providing Git branch status, Python virtual environment display, SSH awareness, and exit code rendering.
- **`vscode-family-ide.md`**: Configuration recipes for VSCode-family editors (VS Code, Cursor, Windsurf, Antigravity) — markdown preview by default, Zen Mode without full-screen, keybinding setup, and merge instructions.
